# 💳 SmartPay – Digital Wallet App

A modern fintech web app built with React, TypeScript, and Tailwind.

## 🚀 Features
- User Authentication (Sign In / Sign Up / Forgot Password)
- Add & Manage Cards
- Savings Goals Tracker
- Send & Request Money
- Dark/Light Mode Support
- Interactive UI with animations

## 🛠 Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Radix UI
